# ウォークスルー：トップページ機能カテゴリの再編成

## 変更サマリー

増え続けていた分析機能メニューを整理し、統計的意味合いが分かりやすいようにトップページのカテゴリを再編成しました。

### 変更内容（`index.html`）

| 旧カテゴリ | 新カテゴリ構成 |
|---|---|
| ユーティリティ | 1. ユーティリティ（維持） |
| データの探索・要約 | 2. データの探索・要約（並び順整理） |
| 平均値の差の検定 | 3. 平均値の比較（パラメトリック検定） |
| ノンパラメトリック検定 | 4. 順位の比較（ノンパラメトリック検定） |
| 関連性の検定・予測 | **5. 比率・関連性の検定 ✨新設** |
| 関連性の検定・予測 | **6. 回帰分析・予測モデル ✨新設** |
| 多変量解析・その他 | 7. 多変量解析・その他（維持） |

### 新設・再編されたグループ

**順位の比較（ノンパラメトリック検定）**
- マン・ホイットニーのU検定
- クラスカル・ウォリス検定
- ウィルコクソンの符号付順位検定

**比率・関連性の検定（カテゴリデータの検定）**
- マクネマー検定（※旧ノンパラから移動）
- カイ２乗検定
- フィッシャーの正確確率検定

**回帰分析・予測モデル**
- 単回帰分析
- 重回帰分析
- ロジスティック回帰分析
