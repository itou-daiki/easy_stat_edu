# トップページ機能カテゴリの再編成案

## 課題
現在の `index.html` のグループ分けには以下の課題があります。
1. **「関連性の検定・予測」が混在**: カテゴリカルデータの検定（カイ2乗検定など）と、数値予測（回帰分析）が同じグループに入っており、分かりにくい。
2. **名義尺度の検定の分散**: 新しく追加した「マクネマー検定」（比率の検定）が「ノンパラメトリック検定（順位の比較）」に入っている。

## 改善案
統計的意味合いごとにグループを整理し、以下の **7カテゴリ** に再編成します。

### 1. ユーティリティ
（現状維持：分析準備やデータ整形）
- 分析サポーター / データ加工・整形 / データ結合 / 因子得点算出

### 2. データの探索・要約
（現状維持＋並び順の微修正）
- 探索的データ分析（EDA） / 📊 **クロス集計** / 相関分析

### 3. 平均値の比較 (パラメトリック検定)
（名前を少し明確化）
- t検定 / 一要因分散分析 / 二要因分散分析

### 4. 順位の比較 (ノンパラメトリック検定)
（マクネマー検定を移動し、順位の比較に特化）
- マン・ホイットニーのU検定 / クラスカル・ウォリス検定 / ウィルコクソンの符号付順位検定

### 5. 比率・関連性の検定 ✨【新設】
（名義尺度の検定をこのグループに集約）
- 📊 **マクネマー検定** / カイ２乗検定 / フィッシャーの正確確率検定

### 6. 回帰分析・予測モデル ✨【新設】
（予測やモデリングに特化したグループとして独立）
- 単回帰分析 / 重回帰分析 / ロジスティック回帰分析

### 7. 多変量解析・その他
（現状維持：多数の変数を扱う高度な手法）
- 因子分析 / 主成分分析 / 時系列データ分析 / テキストマイニング

---
以上の再編成で、トップページのUI（`index.html`）を更新します。CSS等の修正は不要です。
承認いただけましたら実装に進みます。
