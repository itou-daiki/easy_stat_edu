# Suggested Commands

## Development
- `npx http-server -p 8765` - Start local dev server
- Open `http://localhost:8765` in browser

## Testing
- `npx playwright test` - Run all tests
- `npx playwright test tests/<name>.spec.ts --headed` - Run specific test with browser

## System (macOS/Darwin)
- `git`, `ls`, `cd`, `rg` (ripgrep) for searching
